# desarrollo-web-avanzado-fimaz-uas
Repositorio de evidencias de la asignatura Desarrollo Web Avanzado – LISI – FIMAZ - UAS

• Objetivo de la práctica 
El objetivo de esta práctica es comprender el uso de clases en PHP, aplicando los conceptos de programación orientada a objetos como encapsulación, uso de atributos privados, constructores y métodos getters y setters.

• Descripción de la clase creada 
Se creó una clase llamada Usuario en el archivo "Usuario.php" la cual contiene dos atributos privados: nombre y correo.  

La clase incluye:
 Un constructor para inicializar los atributos.
 Métodos getters para obtener los valores de los atributos.
 Métodos setters para modificar los valores.

• Instrucciones de ejecución
Ejecutar el archivo index.php en un servidor local mandando a llamar la clase Usuario con un "include".
Abrir el navegador y acceder al archivo `index.php`.
 El programa mostrará en pantalla el nombre y correo del usuario utilizando los métodos getters de la clase.
